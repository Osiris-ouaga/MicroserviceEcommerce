# MicroserviceEcommerce
pour démarrer le projet il faut lancer tout les modules.

En suite aller sur l'URL http://127.0.0.1:8088/

fonctionnalités :
* Liste des produits
* Détails d'un produit
* Ajoute d'un produit dans un pannier
* Passer une commande
